Shallow Copy


A shallow copy creates a new object, but the properties inside it still reference the same memory locations as the original object. If the original object contains nested objects, modifying them will reflect in both copies.

Deep Copy


A deep copy creates a completely independent copy, including all nested objects. Modifications in one object wonâ€™t affect the other.

Hoisting

Function declarations are fully hoisted, meaning you can call the function before its actual declaration.
Variables declared with var are hoisted but only their declarations, not their initial values.
